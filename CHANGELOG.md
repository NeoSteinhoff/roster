# Changelog

## v0.6.0
- Added the motion pass across Roster.
- Added animated count rolls for summary tiles, filters, and tier counts.
- Added a moving roster selection rail and a travel animation when jumping to people.
- Added focused-person context in the sidebar.
- Updated the packaged app version to match the current release.

## v0.5.0
- Added custom groups, related people links, and a fuller contact card.
- Added avatar uploads and richer fullscreen profile views.
- Added group and tag filtering plus better compact mode.
- Rebuilt the app logo and icon set for the app, browser, and Dock.

## v0.4.0
- Renamed the product to Roster.
- Added the desktop app packaging flow with Electron.
- Added local branding, logo assets, app icon, and scale controls.
- Added archive support and CSV import.

## v0.3.0
- Added human-readable memory file export and import.
- Added settings, defaults for new contacts, and keyboard shortcuts.
- Added dark theme improvements, cleaner settings tabs, and more polished data controls.
- Added persistent tags and memory fixes.

## v0.2.0
- Replaced the original demo roster with the user’s real contact list.
- Added tier management, memory capture, note-taking, and local persistence.
- Added Mesh/clay-inspired UI direction, compact view foundations, and relationship health tracking.
